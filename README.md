## Technologies used in the project:
1. HTML
2. CSS
3. JavaScript
4. React
5. React Router
6. Redux
7. Styled Components
8. React Testing Library
9. Additionally the app uses "modalbox-milcz" plugin created and saved beforeheand on NPM.

## To run the project:
1. Download the repo.
2. Run "npm install"
3. Run "npm start"
